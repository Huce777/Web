Sample Description List

TCP
Transmission Control Protocal is a method (protocal) used along
with the Internet Protocal (IP) to send data in the form of message units,
called packets,between computers over the Internet.
IP
Internet Protocal is the method or protocal by which data is sent
from one computer to another on the Internet.Each computer on the Internet is uniquely identified by an IP address.
FTP
File Transfer Protocal is a protocal used to exchange files between computers on the Internet.
HTTP
Hypertext Transfer Protocal is the protocal used for exchanging text,graphic images, sound,video,and other multimedia files on the Web.
